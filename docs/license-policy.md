# License Policy

Spyber is licensed as `AGPL-3.0-only`.

## Intent

If someone modifies Spyber and offers it as a network service, users of that
service should receive access to the corresponding modified source code.

## Practical Rules

- Keep SPDX headers in source files.
- Keep dependency licenses compatible with AGPL distribution.
- Do not copy proprietary crawler rules, datasets, or platform-specific secrets
  into the repository.
- Document third-party data source terms before adding a connector.

## Non-License Limits

The AGPL covers software rights. It does not solve privacy, anti-spam, data
broker, or marketing compliance obligations.
